# MemorIA – Dementia Care & Assistance Platform

## Overview

This project was developed as part of the **PIDEV – 4SAE3 Engineering Program** at **Esprit School of Engineering – Tunisia** (Academic Year 2025–2026).

**MemorIA** is an intelligent full-stack platform designed to support dementia patients, caregivers, and families.
It provides tools for daily assistance, cognitive support, task management, and intelligent prediction using machine learning.

The system is composed of a **frontend**, **backend**, and an **AI service**, ensuring a complete and scalable architecture.
